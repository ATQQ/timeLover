## 技术方案
* 前端：Vue3+TypeScript+Vite
  * 组件库：Vant UI
* 后端：Node+TypeScript
  * 数据库：非关系型文档数据库（暂定）
* 部署：腾讯云Serverless

## 使用

**安装依赖**
```sh
yarn install
```

**开发启动**
```sh
yarn dev
```

**生产构建**
```
yarn build
```
